# Изобретим — прототип STEM-магазина

Магазин развивающих наборов для детей. Флагман — **Enjoy Robotics**, вокруг — партнёрские бренды (роботы, электроника, опыты, конструкторы).

## Как открыть
Просто открой **`index.html`** двойным кликом в браузере. Сервер не нужен — всё работает как есть.

Страницы:
- `index.html` — главная (витрина, подбор по возрасту, флагман, подборки, блог-превью)
- `catalog.html` — каталог с фильтрами (категория, возраст, бренд, «без экрана»)
- `blog.html` — блог для родителей
- `blog/*.html` — отдельные статьи блога
- `pages/first-robot.html` — посадочная «Первый робот» (квиз + гайд + FAQ)

Каталог кликабельный: фильтры работают, ссылки типа `catalog.html?cat=robots` открывают нужный раздел.

## Что можно менять без программиста
- **Название и цвет.** В шапке — текст «Изобретим». Цвет бренда — `--brand` в начале `assets/css/style.css`.
- **Товары.** Все карточки — в `assets/js/data.js` (понятный список, можно править руками).

## Как наполнять каталог (без команды)
Основной путь — **прайс от партнёра**:
```
node tools/parser/parse.mjs prices файл.yml brandId
```
Соберёт черновой каталог из прайса. Категории угадает по названию, статус поставит «проверить».

Запасной путь — **парсер сайта** (для чернового наполнения):
```
node tools/parser/parse.mjs crawl https://site-партнёра.ru/catalog
```
Подробности — в `tools/parser/README.md`. Список партнёров и статусы — в `PARTNERS.md`.

## Дальше по плану
1. Собрать прайсы у первых партнёров → наполнить каталог.
2. Переписать карточки под наш экспертный стиль (возраст, «сам/с родителем», что развивает).
3. Перенести на движок магазина (InSales / AdvantShop) — оплата, доставка, заказы.
4. Заложить маркировку «Честный знак» для детских товаров.

> Это прототип-витрина: показывает, как магазин будет выглядеть и работать. Оплата/заказы — заглушки, подключаются на этапе движка.
